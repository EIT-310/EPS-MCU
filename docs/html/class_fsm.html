void Fsm::DetermineState ( AdcRead::adc_reading  reading)
static

Bedøm en ny target state på baggrund af ADC måling og kald StateChanger()

Batterispændingen bliver sammenlignet med forskællige arbitære spændingsniveauer og en target state bliver besluttet ud fra dette. Herefter betragtes den samlede effekt fra MPPT'erne, og hvis denne ikke overstiger et arbitrært niveau, sænkes target state med 1.

Note
Aloritmen er simpel og mest til for at have et eksempel på en algoritme.

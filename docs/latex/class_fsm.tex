Bedøm en ny target state på baggrund af ADC måling og kald \mbox{\hyperlink{class_fsm_a9c900414072a45b3cf703e3ea2f37c1b}{State\+Changer()}} 

Batterispændingen bliver sammenlignet med forskællige arbitære spændingsniveauer og en target state bliver besluttet ud fra dette. Herefter betragtes den samlede effekt fra MPPT\textquotesingle{}erne, og hvis denne ikke overstiger et arbitrært niveau, sænkes target state med 1.

\begin{DoxyNote}{Note}
Aloritmen er simpel og mest til for at have et eksempel på en algoritme.
\end{DoxyNote}
